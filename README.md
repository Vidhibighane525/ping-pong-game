# 🕹️ Pong Game (JavaScript Canvas)

A simple Pong game built using HTML5 Canvas and JavaScript. This project demonstrates basic game development concepts like animation, collision detection, and user interaction.

---

## 🚀 Features

- Paddle controlled by mouse movement
- Ball movement with collision detection
- Real-time rendering using `requestAnimationFrame`
- Basic physics (bounce mechanics)

---

## 🛠️ Tech Stack

- HTML5
- CSS3
- JavaScript (Vanilla)
- Canvas API
